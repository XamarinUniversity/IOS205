Navigation Patterns

Duration

15 minutes

Tip: If you are doing this exercise live in a session, make sure to make good use of the instructor, they are online to answer any questions you have!

Lab goals

In this exercise, you'll start with an existing Xamarin.iOS project which has serveral isolated View Controllers defined in a storyboard.
Using the Xamarin.iOS designer, you will add a UITabBarController to implement tab navigation.

Required Assets

There is a Exercise 2 folder included with this lab exercise which contains a starter project you will use to begin, assets you'll need to add to your project, and a completed project you can use to check your work.

Steps

This is a individual exercise, intended to be done on your own, use the step-by-step instructions to complete the exercise.

Open the starter project

As a first step, let's open the starter project.

  1. Open the Clock solution in the Exercise 2/Start folder.
  2. Open the MainStoryboard.storyboard file and examine the View Controllers that have been added. The project is identical to the starter project from the previous excercise.

Add a TabBar Controller

Next we'll add a UITabBarController using the iOS Designer.

  1. Add a TabBar Controller from the ToolBox. Notice that it will add three View Controllers: a TabBar Controller and two child View Controllers.
  2. Delete the two child View Controllers by clicking on the grey bar at the bottom of each and pressing the delete key.
  3. Add each screen of the Clock app as children of the Tab Bar Controller. Control-drag from the Tab Bar Controller to the Clock, Stopwatch and About View Controllers and set the relationship to Tab.
  4. And finally, move the Sourceless Segue in the storyboard to point to the TabBar Controller. Run the app.

Adding Text and Images to the Tabs

The TabBar is working but it looks a little plain. Next we'll add images and text to the tabs.

  1. You'll find a collection of icon images in the Exercise 2 Assets folder. Add all 9 images to your project by dragging them into the Resources folder.
  2. Now set the image and title of each tab by clicking on the tab bar item now shown at the bottom of each child view controller. When selected, you'll be able to set the Title and Image properties for each tab. Set appropriate titles and select the corresponding image for each tab. Run the app.

(Bonus) Navigate to the Splits Time screen

Notice that we're not currently able to access the screen that displays our stopwatch's split times. Because we're not using a Navigation Controller, we can't use a Push Segue.
Think of a way to incorporate this screen into the app. The completed solution shows one option.

Summary

Congratulations! You have successfully implemented tab navigation using the TabBar Controller.
